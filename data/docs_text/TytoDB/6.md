# Primary Keys
The primary key is always the first column of any container.

```
+----------------------------------------------------+
|                   PRIMARY KEY                      |
+----------------------------------------------------+
| In every container:                                |
|   - The first column is always the PRIMARY KEY     |
|                                                    |
|   Example:                                         |
|    [ id | name | age | ... ]                       |
|      ^                                             |
|      |                                             |
|   Primary Key (used in indexing & uniqueness)      |
+----------------------------------------------------+
```
